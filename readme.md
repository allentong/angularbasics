jQuery vs Angular Comparison
======================

Works with IE6+ | Works with IE8+
One way binding | Two way bindings
Clean Dom  | Dom Full of Bindings 
No Real Structural Features | Some Structure or none if you please (MV-Whatever)
Library | Framework
